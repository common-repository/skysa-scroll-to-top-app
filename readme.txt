Animated scroll-to-top, button floats unobtrusively at the bottom of your site. A great convenience feature for your visitors.

== Description ==

A Scroll-to-top button for scrolling up to the top of the current page. Customize the label and icon used for the button. When the app button is clicked the page scrolls to the top with a smooth eased animation. The Scroll-to-Top button displays on a Skysa bar at the bottom of your site.

http://www.youtube.com/watch?v=Zj1uGXX2xrk&hd=1
[More Skysa App plugins](http://wordpress.org/extend/plugins/tags/skysa-apps) -|||- [Skysa App Bar Integration plugin](http://wordpress.org/extend/plugins/skysa-official/)

== Installation ==

1. Install the Skysa Scroll-to-Top App either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the plugin.

You can now manage your app settings from Skysa Bar Apps > Scroll-to-Top on your WordPress admin menu.

> ####Optional
> For added style customization and display options, you can install the [Skysa App Bar Integration plugin](http://wordpress.org/extend/plugins/skysa-official/) and connect it to a Skysa account.
